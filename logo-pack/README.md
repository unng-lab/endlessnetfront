# EndlessNet Logo Pack

Готовый пакет логотипа по выбранному первому варианту: знак бесконечности с сетевыми узлами, адаптированный для тёмной и светлой темы.

## Содержимое

- `svg/` — основные векторные версии, надпись преобразована в кривые.
- `png/` — прозрачные PNG нескольких размеров.
- `web/` — favicon, Apple Touch Icon, Android icons, manifest и готовый HTML-фрагмент.
- `social/` — Open Graph изображения 1200×630 для тёмной и светлой темы.
- `source/` — редактируемые SVG с живым текстом.
- `brand/` — палитра и краткие правила использования.
- `preview/` — обзор всех основных вариантов.

Для шапки сайта на тёмной теме используйте:

`svg/horizontal/endlessnet-horizontal-dark.svg`

Для шапки сайта на светлой теме используйте:

`svg/horizontal/endlessnet-horizontal-light.svg`
