Документация

EndlessNet: запуск и эксплуатация

Эта страница описывает пользовательский порядок работы: вход в админку, управление сетями, подключение устройств и публикацию публичной части.

Обзор

EndlessNet помогает командам держать частные сети и подключённые устройства под управлением в одном месте. Публичная часть объясняет продукт, а админка используется для ежедневной работы.

Публичная часть Лендинг, справка и ссылка на вход в рабочий раздел.
Админка Сети, устройства, статусы, подключение и выход из аккаунта.

Админка

Если вход не выполнен, админка открывает страницу входа. После входа доступны:

  • обзор состояния сетей и устройств;
  • создание и просмотр сетей;
  • список подключённых устройств;
  • готовые действия для установки клиента и подключения машины.

Подключение устройства

Подключение выполняется из админки для выбранной сети.

1. Выберите сеть Откройте раздел сетей или устройств и выберите нужное рабочее окружение.
2. Установите клиент Используйте готовую команду установки из раздела “Доступ”.
3. Подключите устройство Выполните команду подключения и проверьте появление устройства в списке.
4. Проверьте статус В админке видно имя устройства, адрес в сети, теги и время последней активности.

GitHub Pages

Публичный сайт, справка и админка публикуются только как статические файлы на GitHub Pages. Сервер приложений не должен раздавать эту статику.

Безопасность

  • Не публикуйте секреты доступа, приватные ключи и служебные параметры.
  • Не добавляйте на публичные страницы временные или локальные способы входа.
  • Все изменения публичной части должны проходить через сборку и публикацию Pages.
